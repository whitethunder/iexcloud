# IEX Cloud

Wrapper for the IEX Cloud API. https://iexcloud.io/docs/api

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'iexcloud'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install iexcloud
